# plasma-ukraine-air-alert
Віджет для відслідковування повітряної тривоги в Україні для Plasma 6.
Статус тривоги оновлюється раз у пів хвилини автоматично, область та регіон можна вписати у код на вибір(
[список](https://ubilling.net.ua/aerialalerts/?raw=true)).
При зміні статусу іконка віджета зміниться на хмарку з знаком оклику. Щоб оновити віджет дотерміново - прошу натиснути віджет та настиснути кнопку "Оновити зараз".
Можлива помилка - Помилка оновлення може залежати від багатьох чинників, в основному - через надмірну кількість запитів, відсутність Інтернету чи інших проблем з сторони користувача.


---
## Встановлення
```bash
git clone https://github.com/YOUR_USERNAME/plasma-ukraine-air-alert
cd plasma-ukraine-air-alert
kpackagetool6 --install .
```


## Вимоги(залежності)


- Plasma 6.x
- Qt 6.x

Використовується відкрите API [ubilling.net.ua](ubilling.net.ua). Дякую [nightflyza](https://github.com/nightflyza) за підтримку цього API.
